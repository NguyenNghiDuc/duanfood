# duanfood

## Yêu cầu chức năng

### Khách hàng

1. Đăng ký tài khoản
2. Đăng nhập / Đăng xuất
3. Xem danh sách món ăn
4. Tìm kiếm món ăn
5. Lọc theo danh mục (đồ ăn, đồ uống, tráng miệng...)
6. Xem chi tiết món ăn
7. Thêm vào giỏ hàng
8. Cập nhật số lượng trong giỏ hàng
9. Xóa món khỏi giỏ hàng
10. Đặt hàng
11. Chọn địa chỉ giao hàng
12. Chọn phương thức thanh toán
13. Xem lịch sử đơn hàng
14. Theo dõi trạng thái đơn hàng
15. Đánh giá món ăn

### Quản trị viên (Admin)

1. Đăng nhập Admin
2. Quản lý món ăn (Thêm, Sửa, Xóa)
3. Quản lý danh mục món ăn
4. Quản lý khách hàng
5. Quản lý đơn hàng
6. Cập nhật trạng thái đơn hàng
7. Thống kê doanh thu
8. Thống kê số lượng đơn hàng
9. Quản lý mã giảm giá
10. Quản lý phản hồi, đánh giá
